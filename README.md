### Local Respawn

Modifies Vintage Story's respawning mechanic to spawn you near your body, rather than the world origin.

Find it here:
https://mods.vintagestory.at/LocalRespawn

For those wanting to build from source, you should be able to build it using Visual Studio Code, which is how I built it (i.e., ran cake build). May be necessary to make a new VS mod template and dragging the files into the new mod, fixing config errors, and building from there.
